# Install Espresso

## Quick Install (Claude Code)

```
/install-plugin mirkobozzetto/espresso
```

That's it. Espresso activates automatically on every session.

## Commands

- `/espresso` — activate espresso mode
- `/espresso off` — deactivate
- `/espresso-rules` — show all formatting rules
- `/espresso-guide` — full token-saving stack setup guide

## Verify

Start a new Claude Code session. You should see "Brewing espresso..." in the status.
Ask Claude anything — responses should be concise, bullet-pointed, no filler.

## Uninstall

```
/uninstall-plugin espresso
```
